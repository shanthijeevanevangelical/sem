# Shanthi Jeevan Evangelical Mission Website

A modern, static website designed for Shanthi Jeevan Evangelical Mission.

## Structure

- **index.html**: Home page with hero section, ministries overview, and impact stats.
- **about.html**: History, vision, and mission.
- **ministries.html**: Detailed information about HIV care, children's ministry, etc.
- **mission-fields.html**: Locations and field updates.
- **impact.html**: Testimonials and detailed statistics.
- **get-involved.html**: Volunteer form and donation info.
- **gallery.html**: Photo gallery with categories.
- **resources.html**: Newsletters and sermons.
- **contact.html**: Contact form and details.
- **css/style.css**: Main stylesheet with variables for colors and typography.
- **js/main.js**: Handles the reusable Header and Footer injection and mobile menu.

## How to Run

Since this is a pure static website with no backend dependencies:

1.  Simply double-click `index.html` to open it in your web browser.
2.  Navigate through the menu to view other pages.

## Customization

-   **Colors & Fonts**: Edit `css/style.css` root variables.
-   **Header/Footer**: Edit the HTML strings in `js/main.js`.
-   **Images**: Replace the Unsplash placeholder URLs in the HTML files with your actual image paths (e.g., `images/photo.jpg`).
